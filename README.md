# MLA Formatting Lab

Interactive MLA formatting practice lab for online Composition courses.

## What students do

Students complete a guided MLA formatting activity, view/save a completion certificate, and open the course MLA Google Docs template.

## Local setup

1. Install Node.js from https://nodejs.org
2. Open this folder in VS Code or another code editor.
3. In the terminal, run:

   ```bash
   npm install
   npm run dev
   ```

4. Open the local URL shown in the terminal.

## Deploy to Vercel

1. Create a free GitHub account.
2. Create a new GitHub repository.
3. Upload all files from this folder into the repository.
4. Create a free Vercel account.
5. Import the GitHub repository into Vercel.
6. Use the default Vercel settings and click Deploy.
7. Copy the Vercel URL into Blackboard.

## Blackboard instructions for students

Complete the MLA Formatting Lab.
Save your completion certificate as a PDF.
Upload the certificate only to the MLA Formatting Lab assignment link in Blackboard.
Then open the MLA Essay Template and click the blue **Use Template** button.
